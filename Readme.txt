Line Follower Robot
This project is a RP2040 based line follower robot.

Components
Maker Pi RP2040 : Simplifying Robotics with Raspberry Pi® RP2040 (comes with driver built in)
Smart Robot Car Chassis Kit  (comes with 2 3v-6v motors)
tracking sensor TCRT5000 x3
40P 20cm dupont wire male to female

Description
The robot follows a black line,makes turns and follows a loop,additionally stops if all sensors see the black line
